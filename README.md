#  Nouvil - Parallax Landing Page

Nouvil is a creative and interactive landing page built with **HTML, CSS, and JavaScript**.  
The project features a beautiful parallax scrolling effect where different layers of images move dynamically while scrolling, creating an immersive and modern visual experience.

This project focuses on practicing advanced front-end concepts such as DOM manipulation, scroll animations, CSS positioning, and responsive design.

---

##  Features

-  Modern responsive landing page
-  Beautiful parallax scrolling effect
-  Animated stars movement
-  Dynamic moon movement
-  Moving mountains and landscape layers
-  Animated boat movement while scrolling
-  Smooth navigation hover effects
-  Dynamic background color changes on scroll
-  Animated title scaling effect
-  Clean and simple user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Scroll Events

---

##  Design Features

The project uses modern CSS techniques to create an immersive experience:

- Dark gradient background theme
- Full-screen hero section
- Layered images using absolute positioning
- Smooth transitions and animations
- Navigation hover effects
- Image layering with `object-fit`
- Moon lighting effect using `mix-blend-mode`
- Responsive layout for different screen sizes

---

## Responsive Design

The website is fully responsive and works on:

Responsive improvements include:

- Mobile-friendly navigation
